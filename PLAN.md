# 新作ノベルゲーム企画・設計書（仮称：プロジェクト・フール）

## 1. プロジェクト概要
* **ジャンル：** メタ構造・日常侵食型ノベルアドベンチャーゲーム
* **テーマ：** タロットカードの大アルカナ22枚が描く「愚者の旅（The Fool's Journey）」の現代的・批判的解釈
* **ターゲット層：** ARG（代替現実ゲーム）ファン、ミステリー・アドベンチャーゲーム愛好家、心理学やタロットの象徴哲学に関心がある層

## 2. コア・コンセプトとアートメッセージ
### 「運命は盲信するものではなく、自ら変えるものである」
本指針の核となるメッセージは、**「用意された運命（カードの意味）への盲信、一喜一憂、および依存は、人間の思考を放棄させる『最大の愚行』である」**という点にあります。

大アルカナの22枚は通常, 無垢な「愚者」が人生の旅を経て精神的完成に至るまでの「魂の成長物語」として肯定的に描かれます。しかし本作では、タロットの世界を学び、その意味に従おうとするプロセスそのものが、プレイヤーを**「愚者（正位置：無限の可能性）」から「本当の愚か者（逆位置：無計画、無責任、現実逃避、他者への盲信）」**へと退化させていく皮肉な二重構造を設計します。

## 3. ストーリー構造（三幕構成）
旅のプロセスは、タロットの3つの進行フェーズ（意識、無意識、超意識）に沿って展開します。

### 【第1幕】無垢なる愚者の誕生と、運命への「従属」
* **対応カード：** プロローグ（カードとの出会い）、0：愚者 〜 VII：戦車
* **プロット：**
  プレイヤーは街角で奇妙なカードを拾い、その裏に書かれた住所を頼りに占い店「The Fool's Journey」にたどり着くことから始まります。そこで占い師「ソフィア」と出会い、拾ったカードが「0：愚者」であることを知らされます。
  ソフィアは「無知」という言葉の多面性（「何も知らない愚行」と「先入観なく未来を楽しめる無限の可能性」というタロットの複数解釈性）について説明し、プレイヤーにタロットに興味があれば旅路を教えてあげると提案します。
  旅の中で「魔術師」や「教皇」といった絶対的なルールや知恵を授ける他者と出会います。提示される「タロットの教え」に従うことでゲームを有利に進めることができるため、プレイヤーはいつしか「自分で考えること」を止め、カードの導き（システム）を盲信する従順な信者になっていきます。外的な勝利を示す「戦車」のカードで第1幕は華々しく幕を閉じます。

### 【第2幕】内省、依存の極致、および「崩壊」
* **対応カード：** VIII：力 〜 XVI：塔
* **プロット：**
  「隠者」の孤独な探索や「吊られた男」での視点転換といった精神的な試練に入ります。
  物語が不穏さを増すなか、プレイヤーはさらにカードの占いにすがり、「悪魔（XV）」が象徴する「甘い幻想や依存、囚われの罠」に完全に絡め取られます。そして、「カードの指示通りに生きてきたはずなのに、すべてが最悪の形で瓦解する」という「塔（XVI）」の激変と崩壊を突きつけられます。ここでプレイヤーは、自らの思考放棄（＝本当の愚行）に気づかされます。

### 【第3幕】運命の超越と「世界」の再獲得
* **対応カード：** XVII：星 〜 XXI：世界
* **プロット：**
  崩壊の逆境から、かすかな希望「星」を見出し、欺瞞と不安が渦巻く「月」の深層心理を突破します。
  真のハッピーエンド「世界（XXI）」に至るためには、これまでのゲームシステム（提示されるタロットの選択肢）そのものを自らの意志で「否定・拒絶」しなければなりません。運命のプログラムから脱却し、初めて「自分の意志で行動を選択した」瞬間に、プレイヤーは真の全体性（ユング的な自己実現）を獲得します。

## 4. ゲームシステムと演出手法
前作（日本オグパシー協会）で用いた「現実を巻き込む演出技術」と「プレイヤーの認知ハッキング」を、本作向けにチューニングして実装します。

1. **クイズアプリ風「選択肢（イリュージョン・オブ・チョイス）」システム**
   * スマートフォンでの操作性に特化し、レイアウト崩れを完全に排除するため、従来のカードめくり操作を廃止。クイズアプリのようなシンプルな縦並びボタン（`[A]`, `[B]`, `[C]`...）による選択肢システムを採用しました。
   * 1周目（従属ルート）では「いつもの徒歩ルートで帰る」などのプレイヤーの能動的な選択肢を提示しますが、どれを選んでも現実のアクシデントなどの強制イベントによって、最終的にタロットの示す運命（カードのいいなり）へと引きずり込まれる「イリュージョン・オブ・チョイス（選択の錯覚）」を表現しています。
   * 選択肢決定後は、古いログがクリアされ、ウィンドウ上部に引いたタロットカードの絵柄が大きく表示され、その下に結果がタイピングされる「ギャルゲー（ビジュアルノベル）風の1画面ダイアログ」仕様にすることで、スクロールのストレスなくビジュアルとストーリーを楽しめる設計にしました。

2. **甘美な「占い師（案内役）」による認知誘導**
   * ゲーム内にプレイヤーを優しく肯定するナビゲーターを配置します。
   * バッドエンドや窮地に陥った際、ナビゲーターは「これは『死神』による新しい始まりの準備だから問題ない」「『愚者』の逆位置は、今は焦らず待つべきという優しい警告である」といった都合の良い「解釈のすり替え」を行い、プレイヤーをより深い依存（＝破滅）へと誘い込みます。

3. **システムへの反逆（物理的・感覚的ギミック）**
   * トゥルーエンドへ到達するためには、ゲーム画面上に提示された「タロットの選択肢」を物理的・システム的に無視（または破壊）するアクションが求められます。
   * 例えば、提示された強制執行のカードをめくるのではなく、Pointer Events を利用したドラッグ操作で、画面下の「DISCARD ZONE（ゴミ箱）」へ捨て去ることで、運命の呪縛を力ずくで引きちぎる感覚を物理的に体験させます。
   * また、2周目の自律選択肢では、単純なボタン選択ではなく、「心の鏡」「拒絶」といった象徴キーワードの石碑オブジェクトをカード上に直接ドラッグ＆ドロップさせることで、プレイヤー自身の意志決定を物理的に表現します。

## 5. 最近の設計・演出変更

1. **話者維持機能と動的な背景色変更（プレイヤー用カラー）の導入**
   * シナリオテキスト内に `[speaker:話者名]` タグを記述することで、ページをまたいで動的に話者を切り替え・維持する設計を導入しました。これにより、冗長なタグ記述を排除しています。
   * 話者が「プレイヤー」の際には、対話ウィンドウのスタイル（背景およびボーダー）が動的に青系（未来・デジタル・意志を示す青）へ変化する仕組みを構築しました。これにより、プレイヤー自身の発言を視覚的に強調し、占い師（ソフィア）や地の文（ナレーション）との区別を明確にしています。

2. **チャットビューおよびTalkビューでの自動進行（空desc遷移）**
   * 選択肢実行後の結果テキスト（`desc`）が空文字の選択肢（例：チャット画面での「ソフィアに画面を見せる」、ノベル画面での「扉を開けて入店する」など）について、不要な空のダイアログ（デバッグ表記だけの空欄）や「トークを進める」ボタンを表示させず、選択決定後に**自動で次のステップへ遷移する（自動進行）**設計を採用しました。これにより、無駄な逆転待機時間をなくし、シーン切り替えのテンポを最適化しています。

3. **デイリー・トランジション（場面切り替え演出）の導入**
   * 日付が変わるステップ（`〇日目`）の開始時に、ゲーム画面全体を暗転（`#05040a`）させ、中央に日付とアルカナ名（例: "2日目", "I : THE MAGICIAN"）をフェードイン＆ズームアニメーションで表示させるアイキャッチ演出を導入しました。
   * トランジションが完了するタイミングと同期してテキストのタイピングが開始されるように制御し、別の日になったというイメージを直感的に分かりやすくしています。

4. **動的背景切り替え機能（`[bg:画像名]`タグ）と日常アセットの追加**
   * 自宅での朝のシーンや、出社後のオフィスのシーン、夕方の街中など、ストーリーの文脈に合わせて背景画像を切り替える機能を追加しました。本文中に `[bg:画像名]` タグを記述するだけで自動的に背景が切り替わり、テキストからはタグが除去されます。
   * 新しい日常背景画像アセットを 3点（`home_morning.png`、`modern_office.png`、`city_street.png`）を生成・配置し、占い店以外のシーンの臨場感を大幅に向上させました。

5. **1日の終わりの内省演出（運命への依存の深化）**
   * 1日の終わりには、自宅に戻ってカードの意味を深く噛みしめる振り返りナレーションを挿入しました。
   * 「今日こんな事があって、アプリにしたがって、このカードの意味はこうだった、だからこの行動をしたことでうまくいったんだな、明日が楽しみだ。」という独白を流すことで、プレイヤーが自律的な思考を放棄し、アプリの運命ナビゲーションへ急速に依存していく心理的変化をストーリー的・演出的に補強しました。

6. **ソウルカード算出時の派手なサイバー数秘術アニメーション**
   * 生年月日入力後の計算プロセスについて、ゲームへの没入感を高める派手なアニメーションを導入しました。
   * 入力された数字がデジタルスロットのように高速でブラー回転し順次ロック。ロックされた数字が光の尾を引いて下の計算式へとフライト移動し、着弾時にスパークと波紋が発生します。
   * 累積の途中計算がリアルタイムで更新され、一次合計が22以上の場合は警告バナーとともに数字の桁同士が左右から衝突・スライドして合体し `3+3=6` へ変化する縮退演出を行います。
   * 最終決定時には魔法陣が超高速回転し、巨大なゴールドパルス文字が脈動したのち、全画面にゴールドフラッシュが走りカード提示へと遷移します。

7. **ケルト十字スプレッドの完全自動一括展開化（スマホ操作性とテンポの向上）**
   * スマートフォンなどの狭い画面でのドラッグやクリック操作の煩雑さを解消するため、手動でのカードめくり操作を完全に廃止しました。
   * 最初のセリフを読み終えて画面をタップすると、10枚のカードが1枚ずつ自動的に（タップなしで）連続して盤面スロットへ滑らかに飛んでいき、ケルト十字が自動構築される演出を導入しました。
   * 1枚配るごとに画面を遮っていた拡大モーダルや解説タイピングを排除し、「セリフの表示とカードの演出」を完全に分離。全ての配置が完了した後に、ソフィアが完成した盤面全体を見せながら解説や契約決済の対話を始めるすっきりとしたテンポに刷新しました。

8. **テキストウィンドウのレイアウト向上とスマホ向けレスポンシブ最適化**
   * スマートフォンの狭いダイアログ内でスクロールバーが表示されるのを防ぎ、美観を損なわないよう調整しました。
   * `.text-content` でのブラウザ標準のスクロールバーをCSSで完全に非表示化し、スマホ表示幅では盤面の最大高さを抑えてダイアログボックスの表示領域を確保。フォントサイズや余白を Talk ビューと同様に十分な高さで統一し、窮屈さのない美しいレイアウトにしました。
   * 盤面のカードの上やデッキの上がタップされた場合でも、タップイベントが透過して進行するようクリック除外判定を最適化しました。
