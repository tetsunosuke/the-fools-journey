# 新作ノベルゲーム企画・設計書（仮称：プロジェクト・フール）

## 1. プロジェクト概要
* **ジャンル：** メタ構造・日常侵食型ノベルアドベンチャーゲーム
* **テーマ：** タロットカードの大アルカナ22枚が描く「愚者の旅（The Fool's Journey）」の現代的・批判的解釈
* **ターゲット層：** ARG（代替現実ゲーム）ファン、ミステリー・アドベンチャーゲーム愛好家、心理学やタロットの象徴哲学に関心がある層

## 2. コア・コンセプトとアートメッセージ
### 「運命は盲信するものではなく、自ら変えるものである」
本作の核となるメッセージは、**「用意された運命（カードの意味）への盲信、一喜一憂、そして依存は、人間の思考を放棄させる『最大の愚行』である」**という点にあります。

大アルカナの22枚は通常、無垢な「愚者」が人生の旅を経て精神的完成に至るまでの「魂の成長物語」として肯定的に描かれます。しかし本作では、タロットの世界を学び、その意味に従おうとするプロセスそのものが、プレイヤーを**「愚者（正位置：無限の可能性）」から「本当の愚か者（逆位置：無計画、無責任、現実逃避、他者への盲信）」**へと退化させていく皮肉な二重構造を設計します。

## 3. ストーリー構造（三幕構成）
旅のプロセスは、タロットの3つの進行フェーズ（意識、無意識、超意識）に沿って展開します。

### 【第1幕】無垢なる愚者の誕生と、運命への「従属」
* **対応カード：** 0：愚者 〜 VII：戦車
* **プロット：**
主人公（プレイヤーの分身）は何も持たない白紙の「愚者」として世界に生まれます。旅の中で「魔術師」や「教皇」といった絶対的なルールや知恵を授ける他者と出会います。
提示される「タロットの教え」に従うことでゲームを有利に進めることができるため、プレイヤーはいつしか「自分で考えること」を止め、カードの導き（システム）を盲信する従順な信者になっていきます。外的な勝利を示す「戦車」のカードで第1幕は華々しく幕を閉じます。

### 【第2幕】内省、依存の極致、そして「崩壊」
* **対応カード：** VIII：力 〜 XVI：塔
* **プロット：**
「隠者」の孤独な探索や「吊られた男」での視点転換といった精神的な試練に入ります。
物語が不穏さを増すなか、プレイヤーはさらにカードの占いにすがり、「悪魔（XV）」が象徴する「甘い幻想や依存、囚われの罠」に完全に絡め取られます。そして、「カードの指示通りに生きてきたはずなのに、すべてが最悪の形で瓦解する」という「塔（XVI）」の激変と崩壊を突きつけられます。ここでプレイヤーは、自らの思考放棄（＝本当の愚行）に気づかされます。

### 【第3幕】運命の超越と「世界」の再獲得
* **対応カード：** XVII：星 〜 XXI：世界
* **プロット：**
崩壊の絶望から、かすかな希望「星」を見出し、欺瞞と不安が渦巻く「月」の深層心理を突破します。
真のハッピーエンド「世界（XXI）」に至るためには、これまでのゲームシステム（提示されるタロットの選択肢）そのものを自らの意志で「否定・拒絶」しなければなりません。運命のプログラムから脱却し、初めて「自分の意志で行動を選択した」瞬間に、プレイヤーは真の全体性（ユング的な自己実現）を獲得します。

## 4. ゲームシステムと演出手法
前作（日本オグパシー協会）で用いた「現実を巻き込む演出技術」と「プレイヤーの認知ハッキング」を、本作向けにチューニングして実装します。

1. **「運命のダイアログ」選択肢システム**
   * 選択肢の代わりに画面にタロットカードが裏向きに提示されます。
   * カードをめくることで「正位置・逆位置」に応じたシナリオが展開しますが、これを「ゲームのルール」として繰り返すうちに、プレイヤーは「自分で決断する責任」から逃れ、カード任せの「思考停止（愚者・逆位置）」に陥るよう心理的誘導を行います。

2. **甘美な「占い師（案内役）」による認知誘導**
   * ゲーム内にプレイヤーを優しく肯定するナビゲーターを配置します。
   * バッドエンドや窮地に陥った際、ナビゲーターは「これは『死神』による新しい始まりの準備だから問題ない」「『愚者』の逆位置は、今は焦らず待つべきという優しい警告である」といった都合の良い「解釈のすり替え」を行い、プレイヤーをより深い依存（＝破滅）へと誘い込みます。

3. **システムへの反逆（メタ・謎解きギミック）**
   * トゥルーエンドへ到達するためには、ゲーム画面上に提示された「タロットの選択肢」を物理的・システム的に無視（または破壊）するアクションが求められます。
   * 例えば、選択肢をクリックするのではなく、ブラウザのソースコードから直接「運命を決定している変数（運命の糸）」を書き換える、あるいはゲームを一度「意図的に強制終了する」といった行動によって、運命に抗う演出を行います。
