# PROPOSAL.md
# 「占いをゲームで学ぶ」観点からの改善提案

> このゲームは「タロット占いへの盲信が招く危険」を体験させるメタフィクションであり、
> 同時に「タロットの本来の使い方を学ぶ」ことを目的としている。
> 以下は、その教育的側面をより強化するための提案である。

---

## 🔴 HIGH PRIORITY（物語の教育的整合性に関わる問題）

### 1. ケルト十字スプレッドの「各位置の意味」を教えていない

**現状:**  
step-11 でケルト十字が自動アニメーションするが、各カードが置かれる「位置の意味」を一切説明していない。
カードが並ぶだけで、プレイヤーは「なぜこの位置にこのカードが置かれるのか」を知ることができない。

**現在の CELTIC_CARDS_DATA の `pos` フィールドには位置名が定義されているが、ゲーム内で表示されていない:**
```
1. 現在の状況 / 2. 直面する障害 / 3. 意識していること
4. 潜在意識・根底 / 5. 過去の要因 / 6. 近い未来
7. 君の立場 / 8. 周囲の環境 / 9. 望みと恐れ / 10. 最終結果
```

**提案:**  
ケルト十字アニメーション完了後、各カードをタップすると「この位置は〇〇を表します。ここに引かれた〇〇というカードは……」と解説するセリフを表示する。
ただし、ゲームの文脈（ソフィアが恣意的に解釈している）は維持する。

---

### 2. 逆位置（リバース）の概念説明がない

**現状:**  
story ファイルに `upright: true/false` が存在し、UI上では「正位置/逆位置」と表示される。
しかし「逆位置とは何か」「なぜ向きで意味が変わるのか」という説明がゲーム内に一切ない。

**提案:**  
step-04（魔術師の解説）の中に、逆位置の基本説明を1ページ追加する。

```markdown
「タロットには『正位置』と『逆位置』という概念があるの。
 カードが逆さに出たとき、その意味は内側に向かったり、ブロックされたりするの。
 ただし、逆位置をどう解釈するかは占い師によって考え方が異なる。
 これも、タロットが『答え』ではなく『問い』であることの証よ」
```

---

### 3. 小アルカナのスートと日常の対応が薄い

**現状:**  
step-04 で「ワンド・カップ・ソード・ペンタクル」の4スートを解説しているが、
この知識はそのステップの選択肢 `desc` 内にしか存在せず、
**`correct: false` の選択肢を選んだプレイヤーは全文を読まない可能性が高い。**

また、ケルト十字（step-11）では小アルカナが多数登場するが、
その時点でスートの説明がなくなっている（step-04 から間が空きすぎている）。

**提案:**  
- step-04 の本文（フロントマター外のbody）にもスートの簡易説明を追加する
- または、ケルト十字の各カード表示時に「ワンド＝情熱・行動」などをさりげなく表示する

---

## 🟡 MEDIUM PRIORITY（学習の連続性・整合性の問題）

### 4. 登場するアルカナに空白が多い

**現状:**  
ゲーム内で実際に「解説付きで」登場する大アルカナ：

| ステップ | カード | 解説あり |
|---|---|---|
| step-01 | 0: 愚者 | ✅ |
| step-04 | I: 魔術師 | ✅（詳細） |
| step-06 | II: 女教皇 | △（テーマ名のみ） |
| step-07 | III: 女帝 | △ |
| step-08 | V: 法王 | △ |
| step-09 | VI: 恋人たち | △ |
| step-11 | Celtic Cross | ❌（位置解説なし） |
| step-12 | XI: 正義 | ❌ |
| step-13 | XV: 悪魔 | ❌ |
| step-14 | XVI: 塔 | ❌ |
| step-15 | XXI: 世界 | ❌ |

**IV: 皇帝・VII: 戦車〜XIV: 節制・XVII: 星〜XX: 審判** は
SOUL_CARDS に定義されているが、ストーリー上では一切登場しない。

**提案:**  
- SOUL_CARD 診断結果表示時に、そのカードの「本来の（ポジティブな）意味」を1行追記する
  - 例: `「XV: 悪魔」の本来の意味は『欲望の認識と解放』。執着を自覚することで、人は自由を取り戻せる`
- これにより、22枚すべてのカードに触れる機会ができる

### 5. ソウルカード計算の数秘術的根拠の説明不足

**現状:**  
step-10 で生年月日からソウルカードを計算するが、
「なぜ誕生日を足すのか」「なぜ22で割るのか」という数秘術（ヌメロロジー）の説明がない。
プレイヤーにとってはブラックボックスな「機械的判定」に見えてしまう。

**提案:**  
計算完了後のセリフに、数秘術の簡単な説明を追加する：

```markdown
「数秘術では、誕生日の各数字を足し続けて1〜21の数字を導き出すの。
 これがあなたの魂の旅路の出発点を示す『ソウルナンバー』。
 タロットの大アルカナは、まさにその魂の旅路を描いているのよ」
```

---

### 6. 瞑想モードの素材が「愚者」の1枚しかない

**現状:**  
`MEDITATION_MOTIFS` に定義されているモチーフは `0: 愚者` のみ。
「崖の縁」「白い薔薇」の2つだけ。

**提案:**  
ゲームで登場する各カード（少なくとも I〜VI）に対応するモチーフを追加する。
プレイヤーが発見したカードのモチーフのみ解放される形にすると、
コレクション要素と学習要素が連動する。

## 🟢 LOW PRIORITY（体験の補完・深化）

### 7. 「タロットは占いではなく内省のツール」というメッセージが弱い

**現状:**  
ゲームのテーマは「盲信の危険」だが、
**「では本来のタロットはどう使うべきか」** という正しい使い方の提示がループ2でも曖昧。
loop-2 でソフィアが「カードの象徴だけを提示します。具体的な決断はご自身で」と言うが、
これだけではタロットの教育的メッセージ（内省の道具であること）をプレイヤーに十分に実感させるには至らない。

**提案:**  
- **3周目の展開に「フィロ」を配置する**: 3周目（超意識）では、プレイヤーを依存させるソフィアに代わり、真に寄り添い自律的な思考を促す占い師「フィロ」が登場。フィロは「対話鑑賞」の概念を用いて、カード絵柄の中から気になったパーツ（モチーフ）をプレイヤーに選択させ、その象徴キーワードを提示します。プレイヤーはそこから内省し、自律的な「行動」を選択・決定して運命を自らの手で書き換えます。これは、システムによる運命の決定論的インストールを拒絶し、自らカードの象徴を統合して歩む「本来の愚者の旅（The Fool's Journey）」の体現となります。
- トゥルーエンド到達後のエンディングメッセージに以下を追記する：

```
タロットは「答えを教える道具」ではなく「問いを深める鏡」。
22枚の大アルカナは、人生のすべてのフェーズを象徴している。
あなたは今日、自分の内側に問いかける力を手に入れた。
```

### 8. カードコレクション図鑑に「本来の意味」がない

**現状:**  
カード図鑑でカードを開くと、ゲーム内での「ソウルカード的な歪んだ解釈」が表示される
（例: 「XII: 吊られた男」→「耐えることを美徳とし、システムのために自分を犠牲にする」）。

これはゲームの文脈として正しいが、**プレイヤーが実際のタロットを学ぶ際に誤解を生む可能性がある。**

**提案:**  
SOUL_CARDS の desc に「本来の意味（ポジティブ側面）」を追加し、
図鑑で「ゲーム内解釈 / 本来の意味」を切り替えられるタブを追加する。

```js
// 例
12: {
  name: "XII : 吊られた男",
  desc: "耐えることを美徳とし...",  // ゲーム内（歪んだ）解釈
  trueDesc: "自発的な犠牲と新しい視点。手放すことで得られる深い洞察。"  // 本来の意味
}
```
