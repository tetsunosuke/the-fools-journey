# 「占いをゲームで学ぶ」観点からの改善提案・検討メモ（PROPOSAL）

## 1. 優先度：低（PLANに基づく長期的な拡張方針）

### 1.1 小アルカナの導入と大アルカナアドバイスの仕組み
* **提案内容**:
  - 3周目のシナリオ、または追加コンテンツにおいて、日常の葛藤を小アルカナの2つのスートの二択（例：情熱の「ワンド」 vs 感情の「カップ」）としてプレイヤーに提示します。
  - 選択したスートに対して、大アルカナが「アドバイスカード」として現れ、フィロがそのアドバイスから内省を促すフローを構築します。
  - **期待される効果**:
    - スートの性質（ワンド・カップ・ソード・ペンタクル）の理解と、大アルカナの補助的役割を直感的に学習させることができます。

### 1.2 瞑想モード（Daily Meditation）の小アルカナ対応
* **提案内容**:
  - 大アルカナ全22枚に加え、小アルカナのスートや主要カード（各スートのエースなど）についても、気になるモチーフを選択して内省できる仕組みをDaily Meditationモードに組み込みます。

### 1.3 デイリー内省ツールの実用化（毎日引きたいときに引けるライフツール）
* **提案内容**:
  - ゲームとしてのクリア後のおまけ要素から一歩進め、起動時に直接「Daily Meditation」を実行できるショートカットや、今日引いたカードと選択したモチーフの「内省ログ（カレンダーなど）」をブラウザのLocalStorageに蓄積・閲覧できる機能を実装します。
  - ゲームの枠組みを超え、日常的にプレイヤーの心を整える実用的な「タロット内省ノート」へと昇華させます。
