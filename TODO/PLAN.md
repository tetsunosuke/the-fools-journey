# プロジェクト・フール 統合設計・開発計画書

本ドキュメントは、これまでの開発において作成された `PLAN.md`、`idea.md`、および `PROPOSAL.md` を統合し、現状の仕様と今後の開発計画を整理したものです。

---

## 1. プロジェクト概要

* **ジャンル：** メタ構造・日常侵食型ノベルアドベンチャーゲーム
* **テーマ：** タロットカードの大アルカナ22枚が描く「愚者の旅（The Fool's Journey）」の現代的・批判的解釈
* **ターゲット層：** ARG（代替現実ゲーム）ファン、ミステリー・アドベンチャーゲーム愛好家、心理学やタロットの象徴哲学に関心がある層
* **コア・コンセプト：** **「運命は盲信するものではなく、自ら変えるものである」**
  タロットの世界を学び、その意味に従おうとするプロセスそのものが、プレイヤーを「愚者（正位置：無限の可能性）」から「本当の愚か者（逆位置：無計画、無責任、現実逃避）」へと退化させていく皮肉な二重構造を設計します。

---

## 2. ストーリー構造（三幕構成）

### 【第1幕】無垢なる愚者の誕生と、運命への「従属」
* **対応カード：** プロローグ（カードとの出会い）、0：愚者 〜 VII：戦車 (1〜8日目)
* **プロット：**
  主人公は街角で「0：愚者」のカードを拾い、占い店「The Fool's Journey」の占い師「ソフィア」と出会う。ソフィアに勧められてインストールしたアプリ『The Journey』の提示する「運命の教え」に従うことで、日常のトラブルが不思議と好転していく。プレイヤーは自分で考えることをやめ、カードの導き（システム）を盲信するようになっていく。

### 【第2幕】内省、依存の極致、および「崩壊」
* **対応カード：** VIII：力 〜 XVI：塔 (9〜18日目)
* **プロット：**
  本契約を結び、「力」や「隠者」などの精神的な試練に入る。運命への依存は極限に達し、「悪魔（XV）」の罠に完全に絡め取られたのち、「カードの指示通りに生きてきたはずなのに、すべてが最悪の形で瓦解する」という「塔（XVI）」の激変と崩壊を突きつけられる。ここで最初のループが発生する。

### 【第3幕】運命の超越と「世界」の再獲得（2周目・自律ルート）
* **対応カード：** XVII：星 〜 XXI：世界
* **プロット：**
  ループ直後、強烈な既視感とともに再び物語の開始時点に戻される。プレイヤーは占い師ソフィアやアプリへの不信感から元の占い店を拒絶するものの、依然として「指示に従う快適さ（占いへの中毒性）」から抜け出せず、路地裏を彷徨う。その果てに、自律的な思考を促す占い師「フィロ」と出会う。
  フィロの「対話鑑賞（モチーフ選択）」を通じて、プレイヤー自身が「行動」を能動的に決定し、運命を書き換えていく。自律的な意志決定によって、真の「世界（XXI）」の獲得を目指す。

## 3. 実装・反映済みの仕様

以下の仕様は、すでにシステム（`app.js`、`dom.js`、`puzzle.js` 等）およびストーリーMarkdownに完全に反映されています。

### 3.1 システム・演出ギミック
1. **縦並び選択肢・1画面ダイアログ**: クイズ形式の縦並び選択肢ボタンと、スクロール不要な1画面完結型ビジュアルノベル風ダイアログ。
2. **デイリー・トランジション**: 
   シーン切り替え時に `[transition: メイン | サブ]` タグを用いて黒画面のアイキャッチを挿入。朝は「カード裏面パターン」を表示し、夜はその日引いた「カードの表面画像」が薄く明滅する演出。
3. **カードフォーカス時の比率動的変更**: 
   通常の縦長カード表示に加え、恋人と悪魔の対比画像など（URLに `compare` を含むもの）を表示する際は、自動的に横長サイズ（見切れ防止）にコンテナを拡張。
4. **自動化イベント**:
   - 生年月日による「ソウルカード診断」の派手なサイバー数秘術アニメーション。
   - 占い店での「スリーカード（三枚引き）スプレッド」の完全自動一括展開アニメーション。
5. **警告トーストとアプリバブルのシェイク**:
   不正解選択肢を選んだ際、画面右下のアプリバブルの上に警告トーストをオーバーレイ表示し、バブル本体をシェイクさせる演出（バブル未出現時は画面上部へフォールバック）。
6. **話者維持機能と動的な背景色変更（プレイヤー用カラー）**:
   Markdown内の `[speaker:話者名]` タグによる話者管理。話者が「プレイヤー」の際は、ウィンドウのスタイルが動的に青系へ変化する。
7. **チャットビューおよびTalkビューでの自動進行（空desc遷移）**:
   結果テキスト（`desc`）が空文字の選択肢については、自動で次のステップへ遷移させる。
8. **動的背景切り替え機能（`[bg:画像名]`タグ）**:
   本文中に `[bg:画像名]` タグを記述するだけで自動的に背景が切り替わる（`home_morning`、`modern_office`、`city_street` 等の日常アセット対応）。

### 3.2 1周目（loop-1）ストーリーと学習教育的アプローチ
- **「0：愚者」から「XXI：世界」までの大アルカナ完全実装**：
  - 番号順にすべてのカードを欠かさず登場させるため、「IV : 皇帝」および「VII : 戦車」の日常ストーリーを追加。
  - スリーカード（三枚引き）展開時、各位置（過去・現在・未来）をタップした際に、その位置が表す意味（「過去の要因」「現在の状況」「近い未来」）と引かれたカードの象徴解説を表示する仕様を組み込み。
  - 体験版期限切れで「アプリなしで自力で助けに向かう」を選択した際の即時バッドエンド（逆位置の愚者）の実装。
  - 「XV : 悪魔」にて、「恋人たち（6）と悪魔（15）」の数秘・構図の類似対比をビジュアル比較画像付きで解説するイベントを実装。

## 4. 今後検討・開発する残課題（2周目・自律ルート）

1周目のシステムおよびストーリーが完全に整備されたため、ここからは2周目（自律ルート）の本格構築を開始します。

### 4.1 シナリオ設計とステップの実装（後日検討）
* **目標**: `src/content/story/` 配下の `loop-2-step-00.md` 〜 `loop-2-step-27.md` を順次作成する。
* **各ステップの設計方針**:
  * **2周目オープニングの展開 (Step 00〜03)**:
    既視感によるソフィアの店への拒絶、占い中毒による彷徨、別の占い師「フィロ」との出会い。
  * **指示の消失**:
    フィロの店や2周目のアプリにおいては具体的な命令を表示せず、カードの象徴キーワードのみを提示する。
  * **ドラッグパズルによる決断 (`view: puzzle`)**:
    「自律（Self）」や「制御（Control）」などのキーワードが刻まれた石碑（オブジェクト）をカード上に直接ドラッグ＆ドロップさせることで、プレイヤー自身の意志決定を物理的に表現する。
  * **フィロの対話鑑賞**:
    「この絵柄のどこが気になる？」という問いかけから、選択されたモチーフの象徴に基づいてプレイヤー自身が自発的に行動を選択する対話鑑賞フロー。
  * **メタ反逆とTrue Endingの獲得**:
    提示された強制執行カードをゴミ箱ゾーンへ捨てるドラッグ操作、あるいはステータスそのものを破壊するアクションなど、システムへの反逆を通じて「世界（自己実現）」のトゥルーエンドを獲得する。

### 4.2 トゥルーエンドクリア後の「対話鑑賞（Daily Meditation）モード」
- クリア後に解放されるプレイヤー自身の内省ツールとして、大アルカナ全22枚すべてのカードでモチーフ選択による対話鑑賞が行えるように実装。
